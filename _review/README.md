# PDFの作り方

## カラー版（電子書籍向け）

- $ ./build_pdf.sh
- 表紙、裏表紙は_figures/coversに置いてあるので、PDF作成後にmacのプレビューで画像を差し込んでPDFを完成させます。

## モノクロ版（印刷書向け）
- $ ./build_pdf_for_print.sh
- ノンブル（通し番号）が付きます。
