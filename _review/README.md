# PDFの作り方

## 共通作業

- 最後に余分なページが入るので、macOSのプレビューアプリなどで削除する
- Acrobat Reader で表示してエラーが出ないことを確認する

## カラー版（電子書籍向け）

- $ ./build_pdf.sh

## モノクロ版（印刷書向け）
- $ ./build_pdf_for_print.sh
- ノンブル（通し番号）が付きます。
